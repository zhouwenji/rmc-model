
欢迎访问 RUC-MESSAGEix-China (RMC) 文档！
==========================================

RUC-MESSAGEix-China (RMC) 模型是基于开源建模框架 MESSAGEix （`IIASA ECE Programme, 2020 <../references.html>`_）开发的中国省级能源-经济-环境 (E3) 模型 。该模型由中国人民大学应用经济学院周文戟教授团队与华东理工大学商学院任宏涛教授联合开发。
